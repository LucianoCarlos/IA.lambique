Entre com os dados

Bem vindo ao IA.lambique. Abaixo existe uma área onde é possível arrastar e soltar arquivos para análise. O Arquivo deve ser confeccionado em formato csv onde cada coluna deve conter um parâmetro e nas suas respectivas linhas os seus valores para cada observação. Obrigatoriamente na última coluna deve conter os resultados observados para cada conjunto de parâmetros (Veja o tutorial para mais detalhes). Após essa etapa você poderá desfrutar todos ferramentas que o sistema oferece.
